# Catálogo maestro OAuth 2.0/OIDC

Estructura generada para el entregable web de evaluación de riesgos y controles.

## Archivos

- `data/catalogo/catalogo.json`: metadatos.
- `data/catalogo/escenarios.json`: tres arquitecturas evaluables.
- `data/catalogo/riesgos.json`: riesgos globales y específicos.
- `data/catalogo/controles.json`: controles globales y específicos.
- `data/catalogo/riesgo-controles.json`: relación muchos-a-muchos riesgo-control.
- `data/catalogo/referencias.json`: referencias técnicas.
- `data/catalogo/mapeo-score.json`: valores iniciales para cálculo de score.
- `data/filtros/*.json`: vistas por escenario.
- `data/evaluaciones/ejemplo-spa-bff-idp.json`: ejemplo de evaluación diligenciada.
- `docs/modelo-dominio.md`: DMD en Mermaid.
- `data/catalogo-maestro.json`: catálogo maestro consolidado para carga rápida de la UI o demos.
## Nota

Los costos son rangos estimados de esfuerzo incremental y deben calibrarse si se aplican a un caso real.
